## 今まで経験したプロジェクトで元も成功したと感じたところは？失敗したところは？
社内業務基幹システムの刷新のプロジェクトにおいて、技術を活用し、プログラムの開発工数を大幅に削減できたことです。
SAP R/3と他の業務システム連携のため、R/3から日々16種類のマスタデータをアドオンテーブルから抽出し、マスタファイルに出力することになりました。
これに対して、総計16個のプログラム、1人月の開発工数を見込んだこととなりました。
其々のプログラムの処理が似ているため、テーブル名を指定して各マスタの抽出を実現できれば、一本のプログラムで16個マスタデータの抽出できるのではないか
と考えました。

それに対して、フェルドシンボルの使い方について英語の資料などを調べ、そのようなプログラムを作成しました。
その結果は、16本のプログラムが一本で実現し、開発工数が1人月から7人日に短縮することできました。
